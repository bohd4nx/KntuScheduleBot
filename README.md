# 📚 KNTU Schedule Bot

Telegram-бот для перегляду розкладу занять студентів **Центральноукраїнського національного технічного університету** (ЦНТУ).

## 🎯 Функціональність

- 📅 Розклад на сьогодні
- 📅 Розклад на завтра
- 📊 Розклад на весь тиждень
- 🔄 Автоматичне визначення типу тижня (чисельник/знаменник)
- 🧭 На вихідних кнопка/команда «тиждень» показує вже наступний навчальний тиждень
- 🔗 Відображення посилання на онлайн-заняття (якщо воно вказане в `schedule.json`)

## 🚀 Встановлення та запуск

### Крок 1: Клонування репозиторію

```bash
git clone https://github.com/bohd4nx/KntuScheduleBot.git
cd KntuScheduleBot
```

### Крок 2: Встановлення залежностей

```bash
pip install -r requirements.txt
```

### Крок 3: Налаштування

Створіть файл `.env` на основі `.env.example`:

```bash
cp .env.example .env
```

Відредагуйте `.env` та вкажіть токен бота від [@BotFather](https://t.me/BotFather):

```env
BOT_TOKEN=ваш_токен_бота
```

Дати семестру тепер задаються в `bot/core/constants.py`:

```python
SEMESTER_START_DATE = datetime.strptime("16.02.2026", DATE_FORMAT)
SEMESTER_END_DATE = datetime.strptime("01.07.2026", DATE_FORMAT)
```

### Крок 4: Налаштування розкладу

Відредагуйте файл `schedule.json` та вкажіть розклад вашої групи. (Згодом буде автоматично)

Для дистанційних занять можна додати поле `online_link` у конкретну пару:

```json
"numerator": {
	"subject": "Предмет",
	"teacher": "Викладач",
	"room": "1",
	"online_link": "https://us02web.zoom.us/j/..."
}
```

Якщо `online_link` присутній, бот автоматично додає в повідомлення рядок «Посилання: підключитися до заняття».

### Крок 5: Запуск

```bash
python main.py
```
